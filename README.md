# Walmart Sales Analysis

## Project Overview
This project analyzes Walmart sales data to understand sales trends, seasonality, and store-level performance.

The goal is to derive insights that can support inventory planning, demand forecasting, and business decision-making.

## Business Objective
- Identify sales trends across time
- Analyze seasonality and holiday impact
- Compare performance across stores
- Support data-driven inventory and sales strategies

## Dataset
The dataset includes:
- Store
- Date
- Weekly sales
- Holiday flag

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
